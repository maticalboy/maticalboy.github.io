# vue-openlayers
<br>
在 ``Vue2`` 上使用 ``openlayers``。
本项目为练习案例。


<br>

## 安装
```
$ npm install
```
或者
```
$ npm install --registry=https://registry.npm.taobao.org
```

<br>

### 运行
```
$ npm run serve
```

<br>

### 打包
```
$ npm run build
```

<br>

### 说明
使用了以下框架和库。<br>
- [Vue 文档](https://cn.vuejs.org/)
- [Vue-Router 文档](https://router.vuejs.org/zh/)
- [openlayers 官网](https://openlayers.org/)
